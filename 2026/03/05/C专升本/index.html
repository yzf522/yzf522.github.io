表达式等价

原码，反码，补码

注意：正数的原码，反码，补码都是原本的二进制数（8位）

原码：数的二进制，正数第一位为0，负数第一位为1

反码：负数原码第一位不变，其他取反

补码：负数反码+1

按位与，按位或，按位异或,按位取反

按位与 & ：将左右的值转化为二进制补码，然后进行与的比较，得到的值（优先级8）

按位或 |：将左右的值转化为二进制补码，然后进行或的比较，得到的值（优先级10）

按位异或^：将左右的值转化为二进制补码，然后进行的比较，相同的为0，不同的为1（优先级9）

按位取反~：将右边数的二进制原码全部区反（补码），再-1（反码），除第一位，其他取反（原码）

按位左移，按位右移

按位左移<<：将左边的数的原码左移右边的数的位,正数补0，负数补1（优先级5）

按位右移>>：将左边的数的原码右移右边的数的位,正数补0，负数补1（优先级5）

栈（Stack）：后进先出的 “弹夹”

1. 核心定义与特性（选择填空必考）

  • 核心特性：后进先出（LIFO, Last In First Out）；
  • ✅ 生活类比：弹夹 —— 最后压进去的子弹，第一个打出来。

二、队列（Queue）：先进先出的 “排队”

1. 核心定义与特性（选择填空必考）

  • 定义：只能在队尾（Rear） 插入（入队），在队头（Front） 删除（出队）的线性表；
  • 核心特性：先进先出（FIFO, First In First Out）；
  • ✅ 生活类比：排队买饭 —— 先来的先买，后来的站后面。

2. 循环队列（专升本只考循环队列，解决假溢出）

用数组实现，定义两个指针：

  • data[MaxSize]：存队列元素的数组；
  • front：队头指针，指向队头元素的前一个位置（初始为0）；
  • rear：队尾指针，指向队尾元素的位置（初始为0）。

3. 循环队列核心考点（超级高频，记死公式）

（1）队空 / 队满判断（必考选择填空）

  • 队空：front == rear；
  • 队满：(rear + 1) % MaxSize == front（牺牲一个位置，避免和队空混淆）；
  • ✅ 记忆口诀：队空头尾相等，队满尾 + 1 模 Max 等于头

（2）入队 / 出队操作（记指针变化）

  • 入队：rear = (rear + 1) % MaxSize;（队尾指针后移，取模防越界）；
  • 出队：front = (front + 1) % MaxSize;（队头指针后移，取模防越界）。

4. 队列的典型应用（选择题高频）

  • 广度优先搜索（BFS）：图 / 树的层序遍历；
  • 系统缓冲区：打印机队列、消息队列；
  • 排队场景：银行叫号、食堂排队。

三、栈 vs 队列 对比记忆（表格，考前看一眼）

维度 栈（Stack） 队列（Queue）
核心特性 后进先出（LIFO） 先进先出（FIFO）
操作位置 仅栈顶（Top） 队尾入、队头出
空标志 top == -1（顺序栈） front == rear（循环队列）
满标志 top == MaxSize-1 (rear+1)%MaxSize == front
典型应用 括号匹配、递归 BFS、缓冲区

树与二叉树

一、树的基本概念（选择填空）

  1. 树：一对多的层次结构。
  2. 结点：树上的一个元素。
  3. 度：一个结点有几个孩子，度就是几。
  4. 叶子结点：度 = 0 的结点（没有孩子）。
  5. 树的度：树中最大的结点度。
  6. 深度 / 高度：
    • 根在第 1 层，深度 = 层数。

二、二叉树（考试 90% 都考它）

1.定义

  • 每个结点最多 2 个孩子（左孩子、右孩子）
  • 左、右孩子不能颠倒（有序树）

2. 二叉树 5 大性质（必考！背下来！）

  1. 第 i 层最多有：2^(i-1) 个结点

  2. 深度 k 的二叉树最多：2^k - 1 个结点

  3. 叶子结点数 n0 = n2 + 1

    （叶子 = 度为 2 的结点数 + 1）

  4. n 个结点的二叉树，最小深度：⌈log₂(n+1)⌉例如：n=5，2^2=4 <5+1 <2^3=8,h=3

  5. 完全二叉树，双亲下标：i/2，左孩子：2i，右孩子：2i+1

三、两种特殊二叉树（必认）

1. 满二叉树

  • 每一层都满
  • 总结点：2^k - 1

2. 完全二叉树

  • 只有最后一层不满
  • 缺的结点都靠右
  • 前面每一层都满

四、二叉树遍历（超级重点！必考！）

给你最简单、最不会错的记法：

  • 先序：根 → 左 → 右

  • 中序：左 → 根 → 右

  • 后序：左 → 右 → 根

考试必考结论

  1. 先序 + 中序 可以唯一确定二叉树
  2. 后序 + 中序 可以唯一确定二叉树
  3. 先序 + 后序 不能唯一确定二叉树

五、哈夫曼树（选择题必考）

1. 构造方法

每次选两个最小权值的结点合并，生成新结点。

2. 性质

  • 叶子结点：n 个
  • 总结点：2n - 1
  • 没有度为 1 的结点

3. WPL 带权路径长度

每个叶子的 权值 × 路径长度 全部加起来。
